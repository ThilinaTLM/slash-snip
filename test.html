SlashSnip Test Page

Basic Testing:
  1. Open the SlashSnip popup and create a template with trigger /hello
  2. Type /hello (with a space at the end) in the inputs below
  3. The trigger should expand to your template content
  4. Open DevTools (F12) to see debug logs
Phase 2 - Placeholder Testing:

Create these test templates in the popup:

  • /clip → Clipboard: <clipboard>
  • /today → Date: <date:YYYY-MM-DD>
  • /time → Time: <time:HH:mm>
  • /sig → Best regards,<cursor>\n- Name
  • /full → <date> <time> - <clipboard>

Test cases:

  1. Copy text to clipboard, type /clip - should show clipboard content
  2. Type /today - should show current date (YYYY-MM-DD)
  3. Type /time - should show current time (HH:mm)
  4. Type /sig - cursor should be positioned after "Best regards,"
  5. Type /full - should show date, time, and clipboard content
Phase 3 - Interactive Templates:

Create these test templates for user input, tab stops, and transforms:

  • /name → Hello <input:Name>!
  • /namedef → Hello <input:Name:World>!
  • /color → Color: <select:Pick:red,green,blue>
  • /form → Name: <tab:1>\nEmail: <tab:2>
  • /formdef → Status: <tab:1:TODO>
  • /upper → CLIPBOARD: <clipboard:upper>
  • /lower → clipboard: <clipboard:lower>
  • /title → Name: <clipboard:title>
  • /mixed → Dear <input:Name>,\n<tab:1:message here>\nDate: <date>

Test cases:

  1. /name - dialog should appear asking for Name, value gets inserted
  2. /namedef - dialog should have "World" pre-filled as default
  3. /color - dropdown dialog should appear with red, green, blue options
  4. Press Escape in dialog - expansion should be cancelled, trigger text remains
  5. /form - after expansion, Tab key should navigate between positions
  6. /formdef - "TODO" should be selected, Tab/Shift+Tab to navigate
  7. Copy "hello world" to clipboard, type /upper - should show "HELLO WORLD"
  8. Copy "HELLO WORLD" to clipboard, type /lower - should show "hello world"
  9. Copy "john doe" to clipboard, type /title - should show "John Doe"
  10. /mixed - combines input dialog with tab stops and date placeholder
